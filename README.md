# Mandolin Chords for Ultimate Guitar

Browser extension that adds mandolin chord diagrams (GDAE tuning) to Ultimate Guitar chord pages.

## Install

Load as an unpacked extension in Chrome (`chrome://extensions`) or Firefox (`about:addons`). Currently only tested in firefox.

## Tests

Open in a browser:

- `tests/renderer-tests.html` — chord renderer tests
- `tests/content-tests.html` — content script DOM assumption tests
